# Tips & Tricks

| Tip | How |
|-----|-----|
| **Fastest timer start** | `вҢҳвҢҘвҢғвҮ§M` вҶ’ type 3 chars вҶ’ `Enter` вҶ’ `Enter` вҖ” under 3 seconds |
| **Fastest pause/resume** | `вҢҳвҢҘвҢғвҮ§M` вҶ’ `Enter` вҖ” under 1 second |
| **Continue yesterday's work** | Log вҶ’ Yesterday вҶ’ `Enter` on entry вҖ” starts new timer with same details |
| **Quick-start from Log** | `вҢҳ1` in Log tab вҖ” instantly starts timer on first entry |
| **Check your day** | Left-click menubar icon вҶ’ panel opens with Log tab showing entries + stats |
| **Fix yesterday's hours** | `вҶҗ` in Log вҶ’ navigate вҶ’ `E` вҶ’ edit hours вҶ’ `Enter` |
| **Book past time** | Track вҶ’ select вҶ’ `Tab` (manual mode) вҶ’ type hours вҶ’ `Enter` |
| **Flexible hours input** | All equivalent: `1.5`, `1h 30m`, `1h30m`, `90m`, `1,5` |
| **Tag a ticket** | Type `#JIRA-123` in description вҖ” auto-extracted as a tag |
| **Wrong project?** | `E` to edit вҶ’ click project name вҶ’ search вҶ’ pick correct one. Hours preserved. |
| **Favorites** | вӯҗ Star your daily projects вҖ” they appear first in Track tab |
| **Plan your morning** | Set Default Tab to "Log" вҶ’ see what's planned before starting |
| **Menubar dot colors** | рҹ”ҙ = idle, рҹҹў = running, рҹҹ  = paused вҖ” glance without opening |
| **Timeline from panel** | `вҢҳT` opens the Timeline вҖ” see your full day at a glance |
| **Drag to create** | Drag in empty timeline space to create an entry for that exact time range |
| **App вҶ’ entry** | Right-click any app usage block вҶ’ "Create entry" pre-fills the time range |
| **Automate repeating work** | Create rules for apps/calendar events you use daily вҖ” entries are suggested automatically |
| **Calendar context** | Enable calendar in Settings вҶ’ Timeline to see meetings alongside your time entries |
